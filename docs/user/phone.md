> 要使用邮件服务，需要在 `.env` 文件中配置 短息 服务

### 注册帐号

[POST] /v1/phone/send/register

发送短信注册帐号

| 参数  | 类型     | 说明   | 必选 |
| ----- | -------- | ------ | ---- |
| phone | `string` | 手机号 | \*   |

### 身份验证

[POST] /v1/email/send/auth

发送短信进行身份验证

| 参数  | 类型     | 说明   | 必选 |
| ----- | -------- | ------ | ---- |
| phone | `string` | 手机号 | \*   |
