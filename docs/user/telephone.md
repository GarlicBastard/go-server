### 发送注册帐号短信

[POST] /v1/phone/send/register

发送短信，获取注册码用于注册帐号

| 参数 | 类型     | 说明   | 必选 |
| ---- | -------- | ------ | ---- |
| pone | `string` | 手机号 | \*   |

### 发送身份认证短信

[POST] /v1/phone/send/auth

发送短信，获取验证用户是否真正拥有手机

| 参数 | 类型     | 说明   | 必选 |
| ---- | -------- | ------ | ---- |
| pone | `string` | 手机号 | \*   |


### 发送验证码到用户已绑定的手机号

[POST] /v1/phone/auth/phone

**注意**: 这里需要 token

发送短信验证码至用户绑定的手机号
