### 新增帮助

[POST] /v1/help

| 参数    | 类型       | 说明                                               | 必填 |
| ------- | ---------- | -------------------------------------------------- | ---- |
| title   | `string`   | 帮助标题                                           | \*   |
| content | `string`   | 帮助标题内容，可传 HTML                            | \*   |
| tags    | `string[]` | 帮助的标签                                         | \*   |
| status  | `int`      | 帮助的状态, `1` 激活, `-1` 未激活                  | \*   |
| type    | `string`   | 帮助的类型. `article` 为普通文章, `class` 则为分类 | \*   |

### 修改帮助

[PUT] /v1/help/:help_id

| 参数    | 类型       | 说明                                               | 必填 |
| ------- | ---------- | -------------------------------------------------- | ---- |
| title   | `string`   | 帮助标题                                           | \*   |
| content | `string`   | 帮助标题内容，可传 HTML                            | \*   |
| tags    | `string[]` | 帮助的标签                                         | \*   |
| status  | `int`      | 帮助的状态, `1` 激活, `-1` 未激活                  | \*   |
| type    | `string`   | 帮助的类型. `article` 为普通文章, `class` 则为分类 | \*   |

### 删除帮助

[DELETE] /v1/help/:help_id

### 获取帮助列表

[GET] /v1/help

| Query 参数 | 类型     | 说明                                               | 必选 |
| ---------- | -------- | -------------------------------------------------- | ---- |
| status     | `int`    | 帮助的状态, `1` 激活, `-1` 未激活                  |      |
| type       | `string` | 帮助的类型. `article` 为普通文章, `class` 则为分类 |      |

### 获取帮助详情

[GET] /v1/help/:help_id
