### 上传文件

[POST] /v1/upload/file

Form 表单文件上传, 目前仅支持单个文件上传

| 参数 | 类型     | 说明         | 必选 |
| ---- | -------- | ------------ | ---- |
| file | `Blob` | 要上传的文件 | \*   |

### 上传图片

[POST] /v1/upload/image

Form 表单图片上传, 目前仅支持单张图片上传

| 参数 | 类型     | 说明         | 必选 |
| ---- | -------- | ------------ | ---- |
| file | `Blob` | 要上传的图片 | \*   |
